\chapter{Theoretische Grundlagen}
\section{Oberflächenplasmonen}
Ein \emph{Plasma} sei ein Gas aus freien Ladungsträgern mit Gesamtladung 0 -- so zum Beispiel ein vollständig ionisiertes Gas. Im Rahmen des Drude-Modells der quasifreien Elektronen in einem metallischen Festkörper kann man die bis auf reibungsartige Kräfte freien Leitungselektronen als Plasma betrachten. In einem solchen Elektronenplasma als Medium können sich Ladungsträgerdichteschwankungen als Wellen fortpflanzen. Man nennt eine solche sich fortpflanzende Plasmawelle \emph{Plasmon}. In einem Volumen aus Plasma gilt für eine sich als ebene Welle fortpflanzende Elektronendichteschwankung, dass das erzeugte elektrische Feld stets parallel zum $k_{VP}$-Vektor ist, den man der ebenen Plasmawelle (dem \emph{Volumenplasmon}) zuordnet.
